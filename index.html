SOBRE MIM

E aí, dev! Se você chegou aqui, prazer, sou a Giovanna!

Sou educadora e estou traçando meu caminho na tecnologia. Minha jornada começou na sala de aula, onde descobri o poder transformador do conhecimento. Com o tempo, percebi que a tecnologia poderia ser uma extensão desse propósito, permitindo criar soluções inovadoras, acessíveis e inclusivas.

Hoje, uno minha experiência na educação com o mundo do código, desenvolvendo projetos que colocam as pessoas no centro. Meu foco está na inclusão, na criatividade e no impacto social.

Acredito que cada linha de código pode ser uma ponte para oportunidades, conectando pessoas e abrindo novos caminhos. Mais do que construir sistemas, vejo a tecnologia como uma ferramenta para transformar vidas.

Cada projeto que desenvolvo carrega esse propósito: cuidar, ensinar e inovar.

Meus Projetos

DEV HACKS

O Dev Hacks é um projeto pessoal criado com o objetivo de facilitar o acesso a códigos simples e diretos para desenvolvedores iniciantes. A ideia surgiu de uma necessidade própria como programadora em formação: encontrar soluções rápidas para problemas comuns do dia a dia. A plataforma reúne "hacks" de código com tecnologias como React.js, Tailwind CSS, CSS e outras ferramentas amplamente utilizadas, organizadas de forma intuitiva para consulta rápida.
